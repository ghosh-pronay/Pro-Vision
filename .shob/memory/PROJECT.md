# Project

**Goal:** Full-stack productivity app with ad monetization for Bangladesh users
**Stack:** React 19 + Vite + Tailwind v4 + Convex + Firebase Auth + Shadcn UI + Framer Motion + Recharts + Zustand + Lucide Icons
**Architecture:** SPA with Convex backend, Firebase Hosting deployment
**Key paths:**

- `src/convex/` — Convex backend functions (25 tables)
- `src/components/ads/` — Ad system (AdManager, 5 networks, slots)
- `src/pages/` — Main app pages (Dashboard, Todo, Habits, Expense, Focus, Wellbeing, Goals, Admin, Settings)
- `src/components/` — Shared components (KanbanBoard, ReceiptScanner, etc.)
- `src/i18n/` — Bengali + English translations
- `src/store/` — Zustand state management
