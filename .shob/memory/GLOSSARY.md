# Glossary

- **AdManager** — Orchestrator component rendering ad slots by position (header, footer, sidebar, in-content)
- **AdSlot** — Generic slot with automatic network resolution and fallback
- **Premium** — Users with `isPremium: true` in Convex users table see no ads
- **setupFirstAdmin** — One-time mutation granting first user admin role
- **hasAdmin** — Query checking if any admin exists in system
- **Convex** — Backend-as-a-service with real-time database
- **Firebase Auth** — Authentication provider
- **VITE\_\* env vars** — Ad network credentials loaded at build time
