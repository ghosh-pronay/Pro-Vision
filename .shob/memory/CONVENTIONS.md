# Conventions & Preferences

- Tech stack: React 19 + Vite + Tailwind v4 + Convex + Firebase Auth + Shadcn UI
- Target audience: Bangladesh users (Bengali + English i18n)
- Premium users see NO ads; free users see all ads
- No custom ad analytics — rely on network dashboards
- All network config via VITE\_\* env vars (no hardcoded credentials)
- Admin role check via Convex backend queries
- Build command: `npm run build`
