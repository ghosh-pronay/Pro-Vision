## Findings considered and rejected

- **bg-mesh CSS animations** (PERF-06): The `will-change` on pseudo-elements is a minor optimization; the `prefers-reduced-motion` media query already handles accessibility. Not worth the CSS rewrite.
- **Route-level prefetching** (PERF-07 enhancement): Nice-to-have but adds complexity. The `key={location.pathname}` removal (already in Plan 011) is the high-leverage fix.
- **Convex schema type alignment** (TECHDEBT-09): The schema is well-designed but disconnected from localStorage. This is by-design — the shim is a development convenience. Type alignment happens when real Convex is deployed.
- **Integration test suite** (TEST-08 full): A comprehensive integration test suite is valuable but out of scope for this round. Plan 007 adds the foundation (1 integration test).
